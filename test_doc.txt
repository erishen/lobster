# LangChain 简介

LangChain 是一个强大的框架，用于开发由语言模型驱动的应用程序。它提供了丰富的工具和抽象，使得构建复杂的 AI 应用变得简单。

## 主要特性

1. **链式调用** - 可以将多个组件串联起来，形成复杂的工作流
2. **记忆管理** - 支持对话历史和上下文管理
3. **工具集成** - 可以轻松集成外部工具和 API
4. **向量存储** - 支持多种向量数据库，实现高效的语义搜索

## 应用场景

- 问答系统
- 聊天机器人
- 文档分析
- 代码生成
- 数据分析

## 快速开始

```python
from langchain.llms import OpenAI
from langchain.chains import LLMChain

llm = OpenAI(temperature=0.9)
chain = LLMChain(llm=llm)
response = chain.run("什么是 LangChain?")
```

## 总结

LangChain 是构建 LLM 应用的最佳选择之一，它简化了开发流程，提高了开发效率。
